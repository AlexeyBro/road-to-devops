# Linux

Базовые команды
--------
Выполнить предыдущюю команду, обычно используется в связки
с какой-то новой командой
- !!
Команда для вывода данных в stdout и написание в консоль (аналог print)
- echo

Команды навигации
--------
Вывести рабочую директорию
- pwd (print working directory) 
Посмотреть содержимое директории
- ls (list)
Работает аналогично ls, но добавляет / к папкам
-- l
Показывает содержимое с подробной информацией
-- ls -l
Перейти в папку
- cd (change directory)
Вернуться назад
-- cd ..
Вызов дополнительных опций для любой команды
- ls --help
Посмотреть дерево файлов
- tree

Работа с папками и файлами
--------
Создать файл
- touch
Создать папку
- mkdir
Скопировать файл в папку
- cp {file_name} {folder_name}/{copy file name}
Переместить файл в папку
- mv
Удалить файл
- rm
Удалить все файлы с определенным расширением
- rm *.txt
Удалить все файлы
- rm *
Удалить папку
- rm -rf {folder_name}

Работа из под sudo (super user do)
С помощью sudo можно выполнять команды с правами администратора, например
создать файл, при этом запись в этот файл будет доступна только для администратора
--------
Перейти в режим администратора
- sudo su {admin_name if needed}
Переключится на другого пользователя
- su {admin_name if needed}
Изменить владельца (chown - change owner)
- sudo chown {group:user} {file_name}
Изменить модификацию доступа (chmod - change mode)
- sudo chmod

Модиифкаторы доступа
--------
4 - read = r--
2 - write = -w-
1 - execute = --x
Базовые значения могут складываться
4+2 (rw-) - чтение и запись
4+2+1 (rwx) - чтение, запись и выполнение
Уровни доступа задаются для 3 уровней пользователей:
- владелец русурса
- группа пользователей
- пользователь
Пример:
- sudo chmod 764 {file_name}

Пакетный менеджер
--------
Вызвать пакетный менеджер
- apt-get
Содержит в себе различные команды для управления
- sudo apt-get install {prog_name} - установка
- sudo apt-get remove {prog_name} - удаление
- apt-cache policy {prog_name} - поиск программ
- apt-get upgrade {prog_name} - обновление программ, если название программы 
не указано, то обновляет все

Поиск
--------
Основная команда поиска - find комбинируется с различными опциями
Поиск по файлам по имени с жестким регистром
- find . -type f -name "{file_name}"
Поиск по файлам по имени без учета регистра
- find . -type f -iname "{file_name}"
Поиск по папка по имени
- find . -type d -name "{directory_name}"
В названии файла/директории можно заменять чать текста звездочкой
- find . -type f -name "file.*"
Поиск по файлам по правам доступа
- find . -type f -perm 0664 (0 обязательно перед нужной комбинацией прав)
Поиск по размеру
- find . -size +10M
- find . -size -10M
Найти все значения кроме на примере поиска по файлам
- find . -type f -not -name "{file_name}"

Посмотреть содержимое файла
--------
- cat {file_name}

Поиск по данным внутри файла
--------
Поиск с учетом регистра в {search_text}
- grep "{search_text}" {file_name}
Поиск без учета регистра в {search_text}
- grep -i "{search_text}" {file_name}
Вывести результат с номером строки
- grep -n "{search_text}" {file_name}

Выполнить что-то еще после выполнения первой команды
--------
Опция -exec включает в себя команду и заканчивается {} +
- find . -type f -name {file_name} -exec grep "{search_text}" {file_name} {} +

Сортировка данных внутри файла
--------
Отсортировать текст и вывести в консоль
- sort {file_name}
Отсортировать числа и вывести в консоль
- sort -n {file_name}
Отсортировать числа в обратном порядке и вывести в консоль
- sort -n -r {file_name}
Отсортировать и создать новый файл с отсортированными данными
- sort {file_name} > {sortedFile_name}

Мониторинг и работа с памятью
--------
Детальный просмотр запущенных процессов в режиме реального времени
- top

Детальный просмотр снимка текущих процессов

Архивация данных
--------
Заархивировать папку через tar
- tar cf {archived_folder_name}.tar {folder_name}
Посмотреть содержимое архива
- tar tf {archived_folder_name}.tar
Разархивировать архив
- tar xf {archived_folder_name}.tar
Разархивировать архив и вывести содержимое архива
- tar xf {archived_folder_name}.tar
Сжать размер архива
- xz {archived_folder_name}.tar
Разархивировать сжатый архив
- unxz {archive_folder_name}.tar.xz
Заархивировать папку через zip
- zip -r {archived_folder_name}.zip {folder_name}
Разархивировать папку через zip
- unzip {archive_folder_name}.zip

Bash shell скрипты
--------
Все скрипты в линукс выполняюся с помощью оболочки Shell
Shell - это программа, которая является посредником между пользователем и 
операционной системой, самые распространенные реализации shell
- bash
- zsh
Что делает shell:
- читает введенную в терминал команду
- разбирает ее
- запускает нужную программу
- получает результат
- выводит его обратно в терминал
Любой bash скрипт начинается с обязательной конструкции
- #!/bin/bash
Выполнить bash скрипт
- bash {script_name}.sh
